Paragraphs
============

段落是reST基礎的文字區塊，各個段落以空白隔開。相同的段落必是靠左對齊。(沒有空格，或是一樣多的空格。)

分隔線
======

分隔線由4個 - 符號(或更多)構成，以區分上、下兩部份。

.. code-block:: reST

    上文部份
    
    ----------
    
    下文部份
    
demo:

上文部份
    
----------
    
下文部份


Inline markup
=============

使用內嵌標記非常簡單。格式：

* 一個 \* 號，斜體格式。
* 二個 \* 號，粗體格式。
* 反引號，程式碼片段、範例等。

如下：

.. code-block:: python
    
    這是內嵌的其中一種樣式。*測試測試*
    
    這是內嵌的其中一種樣式。**測試測試**
    
    這是內嵌的其中一種樣式。``測試測試``
